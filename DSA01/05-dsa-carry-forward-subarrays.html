01

Count 'a-g' Pairs

Problem: Given a string of lowercase characters, return the count of pairs (i, j) such that i < j, s[i] = 'a', and s[j] = 'g'.

Examples

StringAnswer
a b e g a g3 — pairs (0,3), (0,5), (4,5)
a c g d g a g4
b c a g g a a g5

Brute Force — O(N²)

java · O(N²)
int count = 0;
for (int i = 0; i < N; i++) {
    if (str[i] == 'a') {
        for (int j = i + 1; j < N; j++) {
            if (str[j] == 'g') count++;
        }
    }
}
// T.C = O(N²)   S.C = O(1)
ProblemFor "aaaaaaaag" (N = 10⁵), the worst case is N²/2 ≈ 5×10⁹ — a guaranteed TLE.
02

Carry Forward Technique

Core IdeaTraverse from right to left. Carry a running count of 'g's seen so far. When you hit an 'a', its contribution to the answer is exactly the number of 'g's to its right — which is the count you're already carrying.

Trace — str = "b c a g g a a g"

i=7 → 'g' : count=1 i=6 → 'a' : ans += 1 = 1
i=5 → 'a' : ans += 1 = 2 i=4 → 'g' : count=2
i=3 → 'g' : count=3 i=2 → 'a' : ans += 3 = 5 ✓
java · O(N) carry forward
int ans = 0, count = 0;
for (int i = N - 1; i >= 0; i--) {
    if (str[i] == 'g') {
        count++;
    } else if (str[i] == 'a') {
        ans += count;   // all g's to the right of this 'a'
    }
}
return ans;
// T.C = O(N)   S.C = O(1)
Pattern RecognitionCarry forward is the right tool whenever you need to count pairs (i, j) where the answer at i depends on elements to the right. Traverse right-to-left, carry a counter, apply it when you find the left-pair element.

What Is a Subarray?

A subarray is a contiguous, non-empty portion of an array. It is defined by a start index si and an end index ei where si ≤ ei.

DefinitionA subarray of arr[0..N–1] is any arr[si..ei] where 0 ≤ si ≤ ei ≤ N–1. It can be uniquely identified by (si, ei) or by (si, length).

arr = [4, 1, 2, 3, –1, 6, 9, 8, 12] — Quiz

CandidateIs Subarray?Why
[4, –1, 6, 9]✗ No4 is at idx 0, –1 is at idx 4 — not contiguous
[9, 8, 12]✓ YesContiguous indices 6–8
[4, 8, 12, 4]✗ NoSkips elements; also 4 doesn't appear twice adjacently
[9]✓ YesSingle element is always a valid subarray
[] (empty)✗ NoEmpty array is NOT a subarray by standard definition
[7, 8, 4]✓ YesContiguous indices 5–7 of [2,4,1,6,–3,7,8,4]

Total Number of Subarrays

For an array of length N, subarrays starting at index 0 can end at N different positions, subarrays starting at index 1 can end at N–1 positions, and so on.

Starting IndexCount
0N
1N–1
2N–2
……
N–11
FormulaTotal subarrays = N + (N–1) + (N–2) + … + 1 = N(N+1) / 2

For N = 7: total = 7 × 8 / 2 = 28 subarrays.

05

Print a Subarray

Given si and ei (with si ≤ ei), iterate from k = si to k = ei and print arr[k].

java
void printSubarray(int[] arr, int si, int ei) {
    for (int k = si; k <= ei; k++) {
        System.out.print(arr[k] + " ");
    }
}
// T.C = O(N)   S.C = O(1)
06

Print All Subarrays

There are N(N+1)/2 ≈ N² subarrays and printing each takes O(N) — total O(N³).

java · O(N³)
for (int si = 0; si < N; si++) {
    for (int ei = si; ei < N; ei++) {
        // subarray arr[si..ei]
        for (int k = si; k <= ei; k++) {
            System.out.print(arr[k] + " ");
        }
        System.out.println();
    }
}
// T.C = O(N³)   S.C = O(1)
07

Min-Max: Smallest Subarray Containing Both

Problem: Given an array of N integers, return the length of the smallest subarray that contains both the maximum and minimum elements of the array.

Key Observations 1. The optimal subarray has exactly one occurrence of min and one of max as its endpoints.
2. Two cases: [min … max] and [max … min].
3. If min == max (all elements equal), return 1.
Carry Forward InsightScan once left-to-right. Carry the last index where min was seen (idx_min) and the last index where max was seen (idx_max). Whenever you update either, compute the distance to the other's last position and take the minimum.
java · O(N) carry forward
int min = arr[0], max = arr[0];
for (int i = 1; i < N; i++) {
    if (arr[i] > max) max = arr[i];
    if (arr[i] < min) min = arr[i];
}
if (min == max) return 1;

int ans = N, idxMin = -1, idxMax = -1;
for (int i = 0; i < N; i++) {
    if (arr[i] == min) {
        idxMin = i;
        if (idxMax != -1) ans = Math.min(ans, i - idxMax + 1);
    } else if (arr[i] == max) {
        idxMax = i;
        if (idxMin != -1) ans = Math.min(ans, i - idxMin + 1);
    }
}
return ans;
// T.C = O(N)   S.C = O(1)
Brute Force O(N³) Carry Forward O(N) Space O(1)
08

Quick Recap

Carry Forward

Traverse right-to-left, carry a running count. When you hit the left element of a pair, add the carried count. O(N) instead of O(N²).

Subarray

Contiguous, non-empty portion of an array. Defined by (si, ei) with si ≤ ei. Empty arrays are NOT subarrays.

Total Subarrays

N(N+1)/2. Derived from N + (N–1) + … + 1 (Gauss sum).

Print All Subarrays

Three nested loops: O(N³). There's no way to do better when printing all, since output size is itself O(N³).

Min-Max Subarray

Carry last-seen indices of min and max. O(N) single pass. Key: optimal subarray always has min/max as endpoints.

When to Use CF

Use carry forward when you need to count or match pairs (i<j) and the j-side value can be precomputed as a running total.
