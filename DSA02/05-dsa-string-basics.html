String & Character

String

A string is a sequence of characters — or equivalently, an array of characters stored contiguously in memory.

"Welcome to Scaler"
"Hello World!"

Character

A character is a single symbol — a letter, digit, or special symbol from the computer's character set (ASCII/Unicode).

e.g., 'a', 'A', '1', '-', '!', '7'
02

ASCII Values & Type Casting

Every character maps to an integer via ASCII. The key offsets to memorise:

CharacterASCIICharacterASCIICharacterASCII
'A'65'a'97'0'48
'B'66'b'98'1'49
'Z'90'z'122'9'57
Key Offset 'a' − 'A' = 32. To convert lowercase→uppercase: subtract 32. Uppercase→lowercase: add 32. You can also XOR with 32 to toggle case.

Type Casting Behaviour (Java)

Java — Type Cast Examples
// 1. char stored → prints character
char ch = 'a';  print(ch);   // output: a

// 2. char stored in int → prints ASCII value
int  ch = 'a';  print(ch);   // output: 97

// 3. arithmetic → int result
int  ch = 'a' + 1; print(ch);  // output: 98

// 4. cast back to char after arithmetic
char ch = (char)('a' + 1); print(ch); // output: b
03

Switch Case — Toggle Case of a String

Given a string of upper and lower case letters, convert each lowercase to uppercase and each uppercase to lowercase.

Examples

"Hello" → "hELLO"
"aDgbHJe" → "AdGBhjE"

Approach 1 — char[] (Optimal)

Mutate characters in a char[] array, then build a new String once at the end. This avoids the O(N²) cost of string concatenation.

Java
String solve(String A) {
    int N = A.length();
    char[] ans = new char[N];
    for (int i = 0; i < N; i++) {
        if (A.charAt(i) >= 'a' && A.charAt(i) <= 'z') {
            ans[i] = (char)(A.charAt(i) - 32);  // to UPPER
        } else {
            ans[i] = (char)(A.charAt(i) + 32);  // to lower
        }
    }
    return new String(ans);
}
T.C O(N)
S.C O(N)
Why Not String Concatenation? ans += char creates a new String object on every iteration → O(N) per append → O(N²) total. Use char[] or StringBuilder instead.
04

Substrings

A substring is any contiguous part of a string, including single characters and the whole string.

Key Rules

✅ Single character is a substring
✅ Whole string is a substring
❌ Empty string "" is NOT a substring
"abc" is a substring of "abcde" but "ace" is not
Count Formula A string of length N has exactly N(N+1)/2 substrings. For "abcd" (N=4): 4+3+2+1 = 10.

All Substrings of "abcd"

abcd abbccd abcbcd abcd
05

Palindrome Check

A string is a palindrome if it reads the same forward and backward: aaaa, abba, abccba.

isPalindrome(str, si, ei) — Two-Pointer

str = "a b m a d a m t a m"
si=2, ei=6 → substring "madam" → isPalindrome = true
Java
boolean isPalindrome(String str, int si, int ei) {
    while (si < ei) {
        if (str.charAt(si) != str.charAt(ei)) {
            return false;
        }
        si++;  ei--;
    }
    return true;
}
T.C O(N)
S.C O(1)

Longest Palindrome Substring

Find the length of the longest palindromic substring in a given string.

Examples

"anmadamm" → ans = 5 ("madam")
"feacabacabgf" → ans = 7 ("acabaca")
"adaebcdfdcbetggte" → ans = 9 ("ebcdfdcbe")

Brute Force — O(N³)

Try all N(N+1)/2 substrings, check each with isPalindrome, track the maximum length.

Java — BF O(N³)
int maxLength = 1;
for (int i = 0; i < N; i++) {
    for (int j = i; j < N; j++) {
        if (checkPalindrome(s, i, j)) {
            maxLength = Math.max(maxLength, j - i + 1);
        }
    }
}
return maxLength;
T.C O(N³)
S.C O(1)

Expand Around Centre — O(N²)

For every index, treat it as a palindrome centre and expand outward. Handle both odd-length (single centre) and even-length (centre between two chars) palindromes.

Java — Expand Around Centre O(N²)
int longestPalindrome(String s) {
    int N = s.length(), ans = 1;
    for (int i = 0; i < N; i++) {
        int f1 = expand(s, i, i);      // odd-length
        int f2 = expand(s, i, i + 1);   // even-length
        ans = Math.max(ans, Math.max(f1, f2));
    }
    return ans;
}

int expand(String s, int l, int r) {
    while (l >= 0 && r < s.length()) {
        if (s.charAt(l) != s.charAt(r)) break;
        l--;  r++;
    }
    return r - l - 1;   // length of palindrome found
}
T.C O(N²)
S.C O(1)
Why r − l − 1? When the while loop exits, l and r have overstepped by one each. The actual palindrome spans [l+1, r-1], so length = (r-1) - (l+1) + 1 = r - l - 1.
07

String Immutability

In Java and Python, String instances cannot be modified. The reference can point to a new String, but the underlying character data is fixed.

The String Common Pool

String Common Pool (Heap)
s1→"Hello"
s2→"Hello"  (same object!)
s3→"Hello"  (s3 = s2)

Java stores string literals in a shared pool to save memory. new String("Hello") forces a new heap object at cost — so avoid it unless necessary.

Why Immutable?

Why Strings are Immutable If strings were mutable, changing one string would silently corrupt all other references pointing to the same memory location. Immutability makes strings safe to share across threads and data structures.
Costly Operation s1 = s1 + "e" creates a brand-new String object every time. In a loop, this gives O(N²) time and O(N) garbage. Use StringBuilder.append() instead.
08

StringBuilder — The Mutable Alternative

StringBuilder is a mutable sequence of characters. Each append() call modifies the same buffer in O(1) amortised time — no new objects created.

Java
StringBuilder sb = new StringBuilder();
sb.append('a');   // O(1) — same buffer
sb.append('b');   // O(1)
sb.append('c');   // O(1)
String result = sb.toString();  // O(N) — one-time conversion

Quiz: How Many String Objects Created?

Starting from "":
"" → "a" → "ab" → "abc" → "abcd" → "abcde" → "abcdef"
Answer: 6 new String objects created for "abcdef"

Quick Recap

String Definition

A sequence (array) of characters. In Java/Python, strings are immutable — the instance cannot change, only the reference can.

ASCII Offsets

'A'=65, 'a'=97, '0'=48. Gap between upper and lower = 32. Subtract 32 to uppercase; add 32 to lowercase.

Case Conversion

Use char[] to build the result in O(N). Never concatenate in a loop — each += creates a new object, giving O(N²) total time.

Substring Count

String of length N has N(N+1)/2 substrings. Empty string is not a substring. Single characters and the full string are.

Palindrome Check

Two-pointer from both ends: if chars match, move inward; if mismatch, return false. O(N) time, O(1) space.

Expand Around Centre

Try each index as palindrome centre. Expand outward while chars match. Handle odd and even length separately. O(N²) time, O(1) space.

Immutability & Pool

Java stores string literals in a shared heap pool. new String() bypasses the pool. Mutation creates new objects — expensive in loops.

StringBuilder

Mutable char buffer. append() is O(1) amortised. toString() is O(N). Always use when building strings incrementally.
